This app demonstrates how the phonegap barcode scanner plugin works.

To scan, click the scan button and point your phone camera at the barcode. It will automatically detect it and display the information.

You can use the source code as a template to incorporate the phonegap barcode scanner plugin into your application and use the http://build.phonegap.com service to compile it.

The source code is available under the MIT license from my github account: https://github.com/jlinoff/PgBarcodeScanner.git.
